Greibach Normal Form

A Step-by-Step Guide to CFG Conversion

What is Greibach Normal Form (GNF)?

A Context-Free Grammar (CFG) is in GNF if all production rules follow one simple format:

A → aα
  • A is a non-terminal variable (e.g., S, X, A)
  • a is a single terminal symbol (e.g., '0', '1', 'b')
  • α (alpha) is a string of zero or more non-terminals (e.g., ε, B, CB, A1A2)

How to Convert CFG to GNF

We must follow a strict set of rules, in order:

  1. Simplify the Grammar

    Remove useless symbols, ε-productions, and unit productions.

  2. Convert to Chomsky Normal Form (CNF)

    All rules must be in the form A → BC or A → a.

  3. Rename Non-Terminals

    Rename all variables to A1, A2, A3, ... An, with the Start Symbol (S) always as A1.

  4. Substitute Productions

    Iterate from i=1 to n, ensuring all rules Ai start with a terminal. This is where we use the main substitution rules.

Rule 1: Simplified Grammar

Before starting, the grammar must be "clean." We must remove:

  • Useless Symbols: Variables that can never be reached from the Start Symbol (S) or can never produce a terminal string.
  • ε-Productions (Null): Rules like A → ε (unless it's S → ε and S is not on the right side of any rule).
  • Unit Productions: Rules with only one non-terminal on the right, like A → B.

Rule 2: Chomsky Normal Form (CNF)

After simplifying, all rules must fit one of two forms. This makes the next steps predictable.

Form 1: Two Non-Terminals

A → BC

Form 2: One Terminal

A → a

Rule 4: The Substitution Rules

We process all productions Ai in order (i=1, 2, 3...). For any rule of the form Ai → Ajα:

Case 1: i < j

Example: A1 → A3BC

Action: Leave it as-is for now. We will fix this later with back-substitution.

Case 2: i > j

Example: A3 → A1BC

Action: Substitute Aj. Replace A1 with all of its productions.

Case 3: i = j

Example: A3 → A3BC

Action: This is Left Recursion. We must eliminate it immediately.

Eliminating Left Recursion (i = j)

If we have a rule with immediate left recursion:

A → Aα | β
(α is the part after A, β is everything else that doesn't start with A)

We replace it with two new sets of rules, using a new variable (A'):

Standard Formula:
A → βA'
A' → αA' | ε

Example 1: The Problem

Convert the following CFG (assumed simplified and in CNF) to GNF:

S → AA | 0
A → SS | 1

Example 1: Solution (Step 1)

Step 1: Rename Non-Terminals

The Start Symbol (S) must be A1.

S = A1
A = A2

The new grammar is:

A1 → A2A2 | 0    (A1 → 0 is already in GNF)
A2 → A1A1 | 1    (A2 → 1 is already in GNF)

Example 1: Solution (Step 2)

Step 2: Process A1 (i = 1)

We check the rules for A1:

A1 → A2A2    (Rule: Ai → Ajα)
  • Here, i = 1 and j = 2.
  • Since i < j, this rule is fine. We leave it as-is for now.

Example 1: Solution (Step 3)

Step 3: Process A2 (i = 2)

We check the rules for A2:

A2 → A1A1    (Rule: Ai → Ajα)
  • Here, i = 2 and j = 1.
  • Since i > j, we must substitute A1.

We replace A1 with its productions (A2A2 | 0):

A2 → (A2A2 | 0)A1 | 1

This gives us the new rules for A2:

A2 → A2A2A1 | 0A1 | 1

Example 1: Solution (Step 4)

Step 4: Eliminate Left Recursion in A2

Our A2 rules have a problem:

A2 → A2A2A1 | 0A1 | 1

This is in the form A → Aα | β where:

  • A = A2
  • α = A2A1
  • β = 0A1 | 1

We apply the formula (using new variable Z or A2'):

A → βZ
Z → αZ | ε

Our new GNF-compliant rules for A2 are:

A2 → 0A1Z | 1Z
Z → A2A1Z | ε

Example 1: Solution (Step 5)

Step 5: Back-Substitution

Our A1 rule (A1 → A2A2) is still not in GNF. We must substitute our new GNF rules for A2.

A1 → (0A1Z | 1Z)A2 | 0

Our Z rule (Z → A2A1Z) is also not in GNF. We substitute A2 again.

Z → (0A1Z | 1Z)A1Z | ε

Example 1: Final GNF Grammar

After all substitutions, this is the final grammar in Greibach Normal Form.

A1 → 0A1ZA2 | 1ZA2 | 0
A2 → 0A1Z | 1Z
Z → 0A1ZA1Z | 1ZA1Z | ε

(Note: The ε-production can be removed by substituting Z in all rules, creating more productions, but this form is the direct result of the algorithm.)

Example 2: The Problem

Convert the following CFG (assumed simplified and in CNF) to GNF:

S → CA | BB
B → b | SB
C → b
A → a

Example 2: Solution (Step 1)

Step 1: Rename Non-Terminals

S must be A1. We can order the rest (e.g., S, C, A, B).

S = A1
C = A2
A = A3
B = A4

The new grammar is:

A1 → A2A3 | A4A4
A4 → b | A1A4     (A4 → b is GNF)
A2 → b              (Already GNF)
A3 → a              (Already GNF)

Example 2: Solution (Step 2)

Step 2: Process A1, A2, A3

Process A2 (i = 2) and A3 (i = 3):

A2 → b
A3 → a

Both are already in GNF. Nothing to do.

Process A1 (i = 1):

A1 → A2A3   (i=1, j=2.  i < j. Leave it.)
A1 → A4A4   (i=1, j=4.  i < j. Leave it.)

A1 is fine for now. We will fix it with back-substitution at the end.

Example 2: Solution (Step 3)

Step 3: Process A4 (i = 4)

We check the rules for A4:

A4 → A1A4    (Rule: Ai → Ajα)
  • Here, i = 4 and j = 1.
  • Since i > j, we must substitute A1.

We replace A1 with its productions (A2A3 | A4A4):

A4 → b | (A2A3 | A4A4)A4

This gives us the new rules for A4:

A4 → b | A2A3A4 | A4A4A4

Example 2: Solution (Step 4)

Step 4: Substitute again in A4

Our new A4 rules are not in GNF yet:

A4 → b | A2A3A4 | A4A4A4

The rule A4 → A2A3A4 has i = 4, j = 2. Since i > j, we must substitute A2.

We replace A2 with its production (b):

A4 → b | (b)A3A4 | A4A4A4

Now, our A4 rules are:

A4 → b | bA3A4 | A4A4A4

Example 2: Solution (Step 5)

Step 5: Eliminate Left Recursion in A4

Our A4 rules still have a problem:

A4 → A4A4A4 | b | bA3A4

This is in the form A → Aα | β where:

  • A = A4
  • α = A4A4
  • β = b | bA3A4

We apply the formula (using new variable Z or A4'):

A4 → bZ | bA3A4Z
Z → A4A4Z | ε

Example 2: Solution (Step 6)

Step 6: Back-Substitution

Our A1 rules are still not in GNF. We substitute our new GNF rules for A2, A3, and A4.

A1 → A2A3 | A4A4

Substitute A2 → b:

A1 → bA3 | A4A4

Substitute new A4 rules (bZ | bA3A4Z):

A1 → bA3 | (bZ | bA3A4Z)A4

This gives our final GNF rules for A1:

A1 → bA3 | bZA4 | bA3A4ZA4

Example 2: Final GNF Grammar

After all substitutions, this is the final grammar in Greibach Normal Form.

A1 → bA3 | bZA4 | bA3A4ZA4
A2 → b
A3 → a
A4 → bZ | bA3A4Z
Z → A4A4Z | ε

(Again, the Z rule is not fully in GNF and would be substituted, but this is the direct result of the algorithm's steps.)
